Introduction to Web Design

Web design is the process of creating websites using different technologies like HTML, CSS and JavaScript. A well designed website should be attractive, easy to use and responsive on all devices.

HTML provides the structure of the webpage while CSS provides styling and layout. JavaScript is used to add interactive features like animations, forms and dynamic content.

Block Level Tags

The paragraph tag is used to write text on a webpage.

This div tag groups different elements together.

Block level elements always start on a new line and take full width.

Text Formatting Tags

Bold Text makes text important.

Italic Text is used for emphasis.

Underline Text highlights the text.

Highlighted Text marks important information.
